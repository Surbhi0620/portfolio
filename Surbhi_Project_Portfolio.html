Customer Churn Prediction
Apr – Jun 2026

An end-to-end churn prediction pipeline on a restaurant customer dataset — from SQL-based feature aggregation through ML modelling and SHAP explainability, to Excel-based reporting that surfaces high-risk customer cohorts for business action.

Python SQL PostgreSQL XGBoost SHAP Scikit-learn Excel Pandas
Key Contributions
Loaded restaurant customer data into PostgreSQL and wrote window functions to compute churn rates by visit frequency band and loyalty program membership; engineered SpendPerVisit and TotalRating as derived analytical features.

06
Customer Segmentation — RFM & K-Means
Apr – Jun 2026

A full-stack customer analytics pipeline on the UCI Online Retail II dataset — RFM scores computed in SQL, clustered in Python via K-Means, validated in Excel, and presented as a Power BI segment dashboard with business campaign recommendations.

Python SQL PostgreSQL Scikit-learn Power BI Excel Pandas DAX
Key Contributions
Computed RFM scores directly in SQL — MAX(InvoiceDate) for recency, COUNT(Invoice) for frequency, SUM(Revenue) for monetary value — and persisted the scored table back to the database.
Read SQL output into Pandas and applied K-Means clustering on scaled RFM features; used the elbow method and silhouette scores to determine the optimal number of segments.
Labelled clusters as Champions, At-Risk, Hibernating, and New Customers; exported assignments to Excel with a pivot summarising average RFM per segment for stakeholder validation.
Built a Power BI report with a donut chart of segment distribution and a recommendation table mapping each cluster to a targeted CRM campaign action.
